# auswertungstool

Auswertungstool zur Auswertung von Wegedaten der App "MobiDiary".

Funktionen:
-Filterung von Wegen
-Vergleich von Wegen

Beöntigte Eingangsdaten:
-Wegedaten der App "MobiDiary" im JSON-Format mit WegeID (und NutzerID)

Inhalt:
-HTML Oberfläche
-JavaScript Script 
-JavaScript Vergleich
